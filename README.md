# Despliegue en Kubernetes - Presentación

##  Descripción

En este repositorio se presenta el desarrollo del laboratorio práctico de despliegue de una aplicación contenerizada utilizando Kubernetes con Minikube. Se evidencia el uso de Pods, Deployments y Services, así como el acceso final a la aplicación web.

##  Contenido de la presentación

La presentación incluye:

* Inicio del clúster Kubernetes
* Creación de Pod
* Implementación de Deployment
* Escalado de Pods
* Exposición del Service
* Aplicación funcionando (Nginx)
* Conceptos clave de Kubernetes

---

##  Tecnologías utilizadas

* Docker
* Kubernetes
* Minikube
* Kubectl
* Nginx
